# PathTracer

A GPU-accelerated path tracer written in Rust using wgpu and WGSL shaders.

## Features

- Physically-based path tracing
- Progressive rendering
- GPU acceleration via wgpu
- Custom WGSL shaders
- Modular rendering pipeline

- Run with: `cargo run`
